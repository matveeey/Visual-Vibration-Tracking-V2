\doxysubsection{Подробное описание}
Абстрактный класс для описания вибрирующей точки 

Объект класса-\/наследника будет иметь возможность добавления новых пространственных и временных позиций, нахождения частоты вибрации точки на кадре на основе этих данных (\mbox{\hyperlink{class_vibrating_point_a005868411bd6b296d992f3dcd79451d7}{Execute\+FFT()}})

А также проверить пересечение курсором области взаимодействия с точкой (метод \mbox{\hyperlink{class_vibrating_point_a342c75a6ac0f9ab85de1a0dd663eab03}{Is\+Interacted()}})
