\doxysubsection{Подробное описание}
Дочерний (от \mbox{\hyperlink{class_vibrating_point}{Vibrating\+Point}}) класс. Предназначен для помещения множества цветных точек на кадр и отслеживания вибрации каждой из них 

Объект класса -\/ \char`\"{}цветная\char`\"{} точка на видео, положение которой отслеживается с течением времени для вычисления частоты вибрации этой точки

Точку можно отрисовать методом \mbox{\hyperlink{class_colored_point_handler_ad16eb2210bc1e9f8983186088cc9a573}{Draw()}} на кадре

Также при взаимодействии с точкой (\mbox{\hyperlink{class_vibrating_point_a342c75a6ac0f9ab85de1a0dd663eab03}{Vibrating\+Point\+::\+Is\+Interacted()}}) будет выводиться гистограмма частот в отдельное окно (\mbox{\hyperlink{class_colored_point_handler_a3c6b8eef49c18add5c4c56d9035c06df}{Draw\+Histogram()}})
